# Palestine Inclusive Education Toolkit

## Overview
This repository contains a structured resource kit for educators and policymakers working with students with disabilities in Palestine. The toolkit is designed to support teacher training workshops across the West Bank and Gaza, promoting inclusive education practices and policy development.

## Goal
To provide a foundational set of resources that can be easily adapted, version-controlled, and collaboratively improved by stakeholders in the Palestinian education sector.

## Target Audience
- Teachers and school administrators in Palestinian schools
- Education policymakers and ministry officials
- Teacher trainers and professional development facilitators
- Researchers and advocates in disability studies and inclusive education

## Intended Contents
The toolkit will include the following components:

1. **Policy Brief Template** – A markdown template for advocating inclusive education policies.
2. **Glossary of Key Terms** – Definitions of essential concepts in English and Arabic.
3. **Best Practices Checklist** – Actionable items for teachers to implement inclusive strategies.

## How to Use
This repository uses GitHub issues for planning, branches for development, and pull requests for review. Each component is tracked as an issue, and contributions follow a standard Git workflow.
